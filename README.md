> **A premium, AI-powered online examination platform designed for modern academic and professional assessments.**

Web Based Automated Examination System is a state-of-the-art full-stack application that transforms traditional testing into a dynamic, secure, and visually stunning digital experience. Featuring a high-end **Glassmorphism UI**, real-time **Learning Analytics**, and **AI-driven Insights**, it represents the next generation of educational technology.

---

## ✨ Key Features

### 💎 Premium User Experience
*   **Glassmorphism Design**: A sleek, dark-themed interface with vibrant glow effects and fluid micro-animations (Framer Motion).
*   **Interactive Dashboard**: Real-time "Learning Velocity" tracking using dynamic AreaCharts (Recharts).
*   **Adaptive Exam Interface**: A distraction-free, high-performance testing environment.

### 🧠 Intelligent Core
*   **Zenith AI Insights**: Context-aware AI feedback that identifies learning gaps and recommends focus areas.
*   **Automated Question Seeding**: Pre-loaded professional assessments for Java, Python, and Cloud Architecture.
*   **JWT Security**: Military-grade stateless authentication for secure student and admin sessions.

### 🛡️ Robust Architecture
*   **Spring Boot 3 + JPA**: A high-performance, scalable backend engine.
*   **Persistent H2 Storage**: Zero-configuration, file-based database that ensures data persistence out-of-the-box.
*   **RESTful Mastery**: Clean, documented API endpoints for seamless full-stack integration.

## 🚀 Getting Started

### 1️⃣ Clone the Repository
```bash
git clone https://github.com/varshini-botla/automated-examination-web-system
cd automated-examination-web-system
```

### 2️⃣ Start the Backend
```bash
cd aetherexam-backend
mvn spring-boot:run
```
*Backend runs on: `http://localhost:8080`*

### 3️⃣ Start the Frontend
```bash
cd aetherexam-frontend
npm install
npm run dev
```
*Frontend runs on: `http://localhost:5173`*
